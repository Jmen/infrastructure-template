# Infrastructure Template

This is a template repository which can be used to start a project with basic infrastructure for development and deployment.

The project will give the follow capabilities:
- Website (Next.js)
- API (Next.js)
- Database & Schema Migrations (Prisma & Postgres)
- Authentication & OAuth Social Login (Auth.js & GitHub Login)
- Tests (Jest)
- CI/CD (GitHub Actions)
- Hosting & Preview Environment (Vercel)
